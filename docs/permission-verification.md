# 权限全链路验证清单

## 场景1 - 新用户未授权
- 打开小程序 → 拒绝授权
- 应仅见「核心速览」3条内容
- 付费模块显示模糊遮罩(paywall-mask)
- 点击付费内容弹出引导

## 场景2 - 授权登录
- 点击微信一键授权登录
- 显示用户昵称/头像
- 身份为「普通用户」

## 场景3 - 付费流程
- 进入「我的」→ 选择「个人版 49元/月」
- 显示微信收款码
- 上传付款截图
- 填写微信号→提交
- 提示「10分钟内审核开通」

## 场景4 - 审核开通
- 管理后台登录 admin/radar2026
- 查看待审核订单
- 审核通过
- 用户刷新小程序→会员身份→全部内容可见

## 场景5 - 会员过期
- 会员到期后重新登录
- 应恢复普通用户身份
- 付费内容不可见

## 场景6 - 本周/历史权限
- 会员用户：正常访问本周/历史
- 非会员用户：显示引导开通
